Product Versions

Use the following macros to display version and status information about a topic:

MacroExpands To
\QtMajorVersionThe major version number based on the $QT_VER environment variable
\QtMinorVersionThe minor version number based on the $QT_VER environment variable
\QtVersionThe value of the $QT_VERSION environment variable
\QtVerThe value of the $QT_VER environment variable
\techpreviewThe statement that a module or class is preliminary and might change

\techpreview

Appends the tech preview link to the brief sentence and adds the topic to the tech_preview group.
