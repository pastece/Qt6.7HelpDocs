CMake Source File Properties in Qt6 DBus

CMake Commands know about the following CMake source file properties:

See also CMake Property Reference.

CLASSNAME

Overrides the default interface class name.

INCLUDE

Adds an include path.

NO_NAMESPACE

Disables generation of a namespaced name.
