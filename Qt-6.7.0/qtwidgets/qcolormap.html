Member Type Documentation

enum QColormap::Mode

This enum describes how QColormap maps device independent RGB values to device dependent pixel values.

ConstantValueDescription
QColormap::Direct0Pixel values are derived directly from the RGB values, also known as "True Color."
QColormap::Indexed1Pixel values represent indexes into a list of available colors, i.e. QColormap uses the index of the color that most closely matches an RGB value.
QColormap::Gray2Similar to Indexed, pixel values represent a list of available gray tones. QColormap uses the index of the gray tone that most closely matches the computed gray tone of an RGB value.
