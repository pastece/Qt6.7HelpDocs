Detailed Description

This type represents a QCalendarPermission, and can be used to check the status of and request access to the user's calendar.

See QCalendarPermission for documentation of the individual properties and detailed usage requirements.

See also QML Application Permissions, Application Permissions, and QCalendarPermission.

Property Documentation

status : Qt::PermissionStatus

This property holds the current status of the permission.

See also QCoreApplication::checkPermission.


Method Documentation

request()

Requests the permission with its current properties.

See also QCoreApplication::requestPermission.
