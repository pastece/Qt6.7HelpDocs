Qt XML

The Qt XML module provides an implementation of the DOM standard for XML.

Note: Qt XML will no longer receive additional features. For reading or writing XML documents iteratively (SAX), use the QXmlStreamReader and QXmlStreamWriter classes. The classes are both easier to use and more compliant with the XML standard.

The Qt XML C++ Classes page gives an overview over the available classes in this module.

Using the Module

Using a Qt module requires linking against the module library, either directly or through other dependencies. Several build tools have dedicated support for this, including CMake and qmake.

Building with CMake

Use the find_package() command to locate the needed module components in the Qt6 package:

 find_package(Qt6 REQUIRED COMPONENTS Xml)
 target_link_libraries(mytarget PRIVATE Qt6::Xml)

See also the Build with CMake overview.

Building with qmake

To configure the module for building with qmake, add the module as a value of the QT variable in the project's .pro file:

 QT += xml
