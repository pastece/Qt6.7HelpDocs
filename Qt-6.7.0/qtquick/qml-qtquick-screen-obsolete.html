Obsolete Members for Screen

The following members of QML type Screen are deprecated. They are provided to keep old source code working. We strongly advise against using them in new code.

Attached Properties

  • logicalPixelDensity : real (deprecated)

Attached Property Documentation

Screen.logicalPixelDensity : real [read-only, deprecated]

This property is deprecated. We strongly advise against using it in new code.

The number of logical pixels per millimeter. This is the effective pixel density provided by the platform to use in image scaling calculations.

Due to inconsistencies in how logical pixel density is handled across the various platforms Qt supports, it is recommended to use physical pixels instead (via the pixelDensity property) for portability.

See also pixelDensity.
