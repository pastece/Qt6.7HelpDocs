Detailed Description

Note: The class uses void * as the type since including the COM-based d3d12.h headers is not acceptable here. The actual types are ID3D12Device * and ID3D12CommandQueue *.

Note: This is a RHI API with limited compatibility guarantees, see QRhi for details.
