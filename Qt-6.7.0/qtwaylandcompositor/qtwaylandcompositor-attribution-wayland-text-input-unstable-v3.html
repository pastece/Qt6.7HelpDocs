Wayland Text Input Protocol, version unstable v3

Adds support for compositors to act as input methods and send text to applications.

Used in the Qt Wayland Compositor, and the Qt Wayland platform plugin.

The sources can be found in qtwayland/src/3rdparty/protocol/text-input-unstable-v3.xml.
