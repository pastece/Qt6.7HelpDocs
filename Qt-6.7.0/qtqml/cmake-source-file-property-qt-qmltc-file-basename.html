QT_QMLTC_FILE_BASENAME

Specifies a non-default .h and .cpp file name.

This property was introduced in Qt 6.3.

Note: This property is in technology preview and may change in future releases.

Use this property to specify a non-default .h and .cpp file name, which helps to resolve conflicting file names.

See also qmltc-cmake.
