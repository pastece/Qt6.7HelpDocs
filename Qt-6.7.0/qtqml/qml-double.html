double QML Value Type

a number with a decimal point, stored in double precision. More...

Detailed Description

The double type refers to a number with a decimal point and is stored in double precision, IEEE floating point format. It's the same as real.

Properties of type double have 0.0 as their default value.

Example:

 Item {
     property double number: 32155.2355
 }

This value type is provided by the QML language.

See also QML Value Types.
