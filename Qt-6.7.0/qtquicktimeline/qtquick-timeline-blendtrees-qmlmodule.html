Qt Quick Timeline Blend Trees QML Types

This module was introduced in Qt 6.7.

To import the QML types into your application, use the following import statement in your .qml file:

 import QtQuick.Timeline.BlendTrees

BlendAnimationNode

A blend tree node that blends between two animation sources

BlendTreeNode

Base class for all blend tree nodes

TimelineAnimationNode

A blend tree source node that plays a timeline animation
