QT_IOS_LAUNCH_SCREEN

Path to iOS launch screen storyboard used by all targets.

This variable was introduced in Qt 6.4.

Note: This variable is in technology preview and may change in future releases.

Note: This variable is used only if targeting iOS.

Specifies the path to an iOS launch screen storyboard file that will be used by all targets within a project.

See also Launch Screens and Launch Images.
