CMake Source File Properties in Qt6 Core

CMake Commands know about the following CMake source file properties:

See also CMake Property Reference.

QT_DISCARD_FILE_CONTENTS

Specifies that the given files should be empty in the resource file system

QT_RESOURCE_ALIAS

Specifies the Qt resource alias for a file in a resource.
