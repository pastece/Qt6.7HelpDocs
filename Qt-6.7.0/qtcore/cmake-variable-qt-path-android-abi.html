QT_PATH_ANDROID_ABI_<ABI>

Set of variables to specify the path to Qt for Android for the corresponding ABI.

This variable was introduced in Qt 6.3.

Note: This variable is in technology preview and may change in future releases.

Note: This variable is used only if targeting the Android platform.

Each variable can be used to specify the path to Qt for Android for the corresponding ABI. The supported ABIs are armeabi-v7a, arm64-v8a, x86, x86_64.

See also QT_ANDROID_ABIS.
