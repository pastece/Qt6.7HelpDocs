Unicode Common Locale Data Repository (CLDR), version v44.1

The Unicode CLDR provides key building blocks for software to support the world's languages, with the largest and most extensive standard repository of locale data available.

Used in Qt Core (QTimeZone, QLocale).

The sources can be found in qtbase/src/corelib/text/qlocale_data_p.h, qtbase/src/corelib/time/qtimezoneprivate_data_p.h, qtbase/src/corelib/time/qhijricalendar_data_p.h, qtbase/src/corelib/time/qjalalicalendar_data_p.h, and qtbase/src/corelib/time/qromancalendar_data_p.h.
