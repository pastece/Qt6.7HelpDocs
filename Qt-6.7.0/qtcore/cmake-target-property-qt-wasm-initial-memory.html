QT_WASM_INITIAL_MEMORY

Internal WebAssembly initial memory.

This property was introduced in Qt 6.2.4.

Note: This property is in technology preview and may change in future releases.

Note: This property is used only if targeting the WebAssembly platform.

Specifies the initial amount of memory to use, in bytes. Using more will cause the browser to copy the old heap into a new one. Translates into the Emscripten compiler setting of INITIAL_MEMORY. QT_WASM_INITIAL_MEMORY must be a multiple of 65536 bytes.

For more information, see Emscripten compiler settings.
