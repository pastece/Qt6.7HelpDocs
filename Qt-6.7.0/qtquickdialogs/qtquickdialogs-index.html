Qt Quick Dialogs

The Qt Quick Dialogs module allows you to create and interact with system dialogs from QML. The module was introduced in Qt 6.2.

Using the Module

The QML types of the module are available through the QtQuick.Dialogs import. To use the types, add the following import statement to your .qml file:

 import QtQuick.Dialogs
