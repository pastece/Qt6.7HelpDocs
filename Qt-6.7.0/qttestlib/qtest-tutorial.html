Qt Test Tutorial

This tutorial introduces some of the features of the Qt Test framework. It is divided into six chapters:

  1. Writing a Unit Test
  2. Data Driven Testing
  3. Simulating GUI Events
  4. Replaying GUI Events
  5. Writing a Benchmark
  6. Skipping Tests

Note: You can build and execute the tests from each chapter using the available source code, which is linked to at the end of each chapter.
