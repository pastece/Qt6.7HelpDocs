Qt for Linux/X11 - Specific Issues

This page contains information about the X11 platforms Qt is currently known to run on, with links to platform-specific notes.

Linux

There are no known problems with using Qt on Linux. For information about the specific compilers and distributions supported, visit the supported platforms page.
