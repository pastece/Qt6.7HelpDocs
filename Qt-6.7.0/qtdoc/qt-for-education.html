Qt for Education

Welcome to learning Qt!

Qt for Education (Qt Edu) is a special licensing package for learning purposes. You can find more information about educational licensing here.

These instructions walk you through the installation of the different Qt Edu packages, as well as applying for the educational license.

  • Qt Edu for Designers - for those interested in creating user interfaces and functional user experiences
  • Qt Edu for Developers - for those interested in developing Qt applications
    • Developing on Raspberry Pi
  • Qt Edu resources

Qt Edu for Designers package contains Qt Design Studio Enterprise, allowing you to import your designs from design tools such as Figma, Sketch, or Adobe XD, and develop them further into functioning user interfaces on your target devices.
