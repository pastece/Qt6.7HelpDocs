The QtEntryPoint Library

QtEntryPoint is a helper library that enables the developer to write a cross-platform main() function on Windows. If you do not use CMake, qmake, qbs, or any other build tool that handles this automatically, then you need to link against the QtEntryPoint library.

The entry point library is statically linked with the application. To avoid issues with static linking and the GNU Lesser General Public License (LGPL), it is licensed both under the commercial licenses, and the BSD 3-clause "New" or "Revised" License.
